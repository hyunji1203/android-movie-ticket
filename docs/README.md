## 기능 목록

- [X] 하나의 영화가 목록에서 보인다.
- [X] 영화의 상영일은 각자의 범위를 갖는다.
- [X] 영화는 주말과 평일 여부에 따라 상영 시간이 달라진다.
- [X] 영화를 선택하면 상세 정보가 나타나며, 예약할 인원을 선택할 수 있다.
- [X] 영화 티켓 한 장은 13000원이다.
- [X] 할인정책이 있다.
  - [X] 무비데이
  - [X] 조조/야간
